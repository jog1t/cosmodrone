# cosmodrone

Multiplayer game scaffold with a Vite + React + Tailwind frontend and a RivetKit backend.

## Workspace

- `apps/web` - React client for the Cosmodrone mission shell
- `apps/server` - Hono + RivetKit backend exposing `/api/rivet/*`

## Commands

- `pnpm install`
- `pnpm dev`
- `pnpm build`
- `pnpm lint`
- `pnpm format`
